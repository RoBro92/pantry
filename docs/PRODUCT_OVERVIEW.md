# Product Overview

Pantry is a pantry management application for households that want reliable inventory tracking, location-aware storage, and a path toward shopping, recipe, and AI-assisted workflows without locking the product to hosted-only assumptions.

## Core Product Shape

- Self-hosted first, with later SaaS deployment paths.
- Multi-household from day one.
- Household-scoped pantry data with server-side tenant enforcement.
- QR-code-driven navigation to specific pantry locations.
- Import pipeline designed for receipts and other structured or semi-structured files.
- AI integration behind a provider abstraction layer.

## Target Outcomes

- Make it easy to answer: what do we have, where is it, what is expiring, and what should we buy.
- Support households with multiple storage zones such as pantry, fridge, freezer, garage, or utility room.
- Keep the self-hosted experience usable without SaaS-only dependencies.
- Avoid architecture dead ends that would block later hosted tiers.

## Initial Audience

- Self-hosters running a single household instance.
- Early adopters comfortable with Docker Compose.
- Households that want structured inventory before advanced AI or import automation is fully built.
